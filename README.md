## Contributing

For convenience there is a Dev Container ([see tutorial](https://code.visualstudio.com/docs/devcontainers/containers#_installation)), otherwise install [mise](https://mise.jdx.dev/) to setup the dev environment:

```bash
mise install
mise run build:wasm
mise run dev
```
